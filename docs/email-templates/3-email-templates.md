## When to Use Which

| Scenario                                    | Use Version                                |
| ------------------------------------------- | ------------------------------------------ |
| First-time wedding planner, emotional buyer | A – Romantic                               |
| "Alcohol Heroes is too expensive"           | B – Numbers                                |
| "Can I use my own alcohol?"                 | C – Insider                                |
| Referral from a wedding venue               | A – Romantic                               |
| Lead from The Knot / WeddingWire            | B – Numbers (they're comparing)            |
| Someone who listed specific cocktails       | C – Insider                                |
| Corporate event lead                        | B – Numbers (adapt to corporate)           |
| Birthday party lead                         | C – Insider (adapt "wedding" → "birthday") |

## A/B Test Plan (if sending at volume)

1. Send all 3 versions to equal segments (33/33/33)
2. Track: open rate, click rate, form completions
3. After 200 sends, crown the winner
4. Key metric: **form completion rate** (not opens — that's vanity)
