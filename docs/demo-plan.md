# FieldLaunch Demo Plan

## Demo Objective
Show the full loop from structured business data to a generated preview website for a service-area business prospect.

## Demo Inputs
- sample roofing business profile JSON
- reusable roofing preview template

## Demo Outputs
- generated static preview site
- deployable Vite app folder
- public preview-ready artifact

## Demo Business
- Summit Ridge Roofing
- Dallas, Texas

## What This Demonstrates
- profile-driven content generation
- reusable local-service site structure
- preview labeling for outbound sales
- repeatable niche template pattern
